# 催化剂 (Catalyst)

回合制网格策略游戏。在元素网格上放置"规则柱"，启动元素连锁反应，累计连锁总数达成胜利。

**Godot 4.7 + GDScript**

## 运行

用 Godot 4.x 打开 `project.godot`，F5 运行。

## 规则

| 牌 | 类型 | 效果 |
|----|------|------|
| 蒸汽化 | 转化 | 水接触熔岩 → 汽，熔岩冷却为岩 |
| 加速生长 | 增殖 | 树相邻蒸汽 → 向空格扩散树 |
| 丛林灭绝 | 灭绝 | 半径内树+草 ≥5 → 清空，也清汽 |
| 岩化 | 转化 | 岩接触树 → 裸土 |
| 草生 | 增殖 | 裸土旁有树 → 空格生草 |
| 采掘 | 转化 | 树相邻矿 → 两者消失，+5 连锁 |
| 草殖 | 增殖 | 草向空格扩散 |

**世界规则**: 汽 2 回合消散 · 草无邻树 2 回合变土 · 裸土 2 回合变岩 · 每回合随机生成水/岩

**催化剂尘**: 每 5 连锁播撒 3 粒尘，尘 4 连通形成团块，团块触到规则柱则整个团块享受规则效果。风每回合推动陈尘。

**胜利**: 连锁累计 ≥100 · **失败**: 连续 10 回合无连锁 或 单一元素超 50%

## 操作

- 左键手牌 → 选中 · 左键网格 → 放置规则柱(耗1能量)
- 右键规则柱 → 撤回(退能) · 执行演化 → 观看连锁
- ? 按钮 → 帮助面板
